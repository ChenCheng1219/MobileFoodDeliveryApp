9 def register(self, email, password, confirm_password): 

10 """ 

11 注册新用户。 

12 

13 该函数接收电子邮件、密码和密码确认作为输入。它执行一系列检查以确保注册有效： 

14 - 验证电子邮件是否在有效格式。 

15 - 确保密码与确认密码匹配。 

16 - 验证密码是否符合强度要求。 

17 - 检查电子邮件是否已经注册。 

18 

19 如果所有检查通过，用户将被注册，他们的电子邮件和密码将被存储在`users`字典中，确认状态设置为False（表示用户尚未确认）。返回成功消息。 

20 

21 参数： 

22 email (str): 用户的电子邮件地址。 

23 password (str): 用户的密码。 

24 confirm_password (str): 用户密码的确认。 

25 

26 返回： 

27 dict: 包含注册尝试结果的字典。 

28 成功时，返回{"success": True, "message": "Registration successful, confirmation email sent"}。 

29 失败时，返回{"success": False, "error": "具体错误消息"}。 

30 """ 

31 if not self.is_valid_email(email): 

32 return {"success": False, "error": "Invalid email format"} # 如果电子邮件格式无效，返回错误。 

33 if password != confirm_password: 

34 return {"success": False, "error": "Passwords do not match"} # 如果密码不匹配，返回错误。 

35 if not self.is_strong_password(password): 

36 return {"success": False, 

37 "error": "Password is not strong enough"} # 如果密码不够强，返回错误。 

38 if email in self.users: 

39 return {"success": False, 

40 "error": "Email already registered"} # 如果电子邮件已经注册，返回错误。 

41 

42 # 如果所有条件都满足，注册用户并返回成功消息。 

43 self.users[email] = {"password": password, "confirmed": False} 

44 return {"success": True, "message": "Registration successful, confirmation email sent"}

46 def is_valid_email(self, email): 

47 """ 

48 根据简单的验证规则检查提供的电子邮件是否有效。 

49 此规则仅检查电子邮件是否包含'@'符号，并在域名部分有一个'.'。 

50 

51 参数： 

52 email (str): 要验证的电子邮件地址。 

53 

54 返回： 

55 bool: 如果电子邮件有效，返回True，否则返回False。 

56 """ 

57 return "@" in email and "." in email.split("@")[-1]

59 def is_strong_password(self, password): 

60 """ 

61 检查提供的密码是否符合强度要求。 

62 强密码定义为至少8个字符长，至少包含一个字母和至少一个数字的密码。 

63 

64 参数： 

65 password (str): 要验证的密码。 

66 

67 返回： 

68 bool: 如果密码强，返回True，否则返回False。 

69 """ 

70 # c.isdigit判断是否为阿拉伯数字 c.isalpha判断是否为字母 

71 return len(password) >= 8 and any(c.isdigit() for c in password) and any(c.isalpha() for c in password) and not any(c.isspace() for c in password)

74 """ 

75 如果用户的电子邮件和密码与存储的凭据匹配，则登录用户。 

76 

77 参数： 

78 email (str): 用户的电子邮件地址。 

79 password (str): 用户的密码。 

80 

81 返回： 

82 dict: 如果登录成功，包含用户资料的字典， 

83 如果登录失败，返回错误消息。 

84 """ 

85 user = self.users.get(email)
